3. CSS: Hard

Using "display: inline-block" will create a line of conent blocks, that will be by default arranged in a row, which will wrap to a new line if the space will be too small. On the other hand, using "display: block" will result in creating series of blocks, each occupying one row, that will all be arranged as a column in a bigger picture.

Use "float: left" or "float: right" on an element to stick it respectively to left or right side of a container. Use "clear: both" to exclude an element from the float property of a superior element.

Thanks to media queries you can create responsive websites that will adjust depending on given screen size of a particular device.
