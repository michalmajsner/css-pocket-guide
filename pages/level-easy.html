1. CSS: Easy

You can add CSS to your HTML file in three ways:
  1. inline
  2. internal
  3. external
To select particular element to style you can use one of many CSS selector:
  • HTML_tag_name - select all HTML elements of that type
  • .class_name - select only elements with specified class name
  • #index_name - select only elements with specified index name (please note that index name should be unique and given to only one element)
  • * - select all elements
  • element_name[value="number"] - select elements with particular value atribute

We can specify color of background or text in multiple ways. It can be for example just plain name of a color, its hex code or RGB values.

In CSS you can not only change the color of the font, but also import new fonts, change their size, weight, align them or capitalize all the letters. It is recommended to use rem units instead of pixels while specifying font size, because the first ones are based on the size preferrably declared in :root element. So if we would want to change font sizes, but keeping their proportions, we change value only in one place and not in multiple sections of our file. We can also refer just to superior element in HTML hierarchy (for example body section) by using em units.
